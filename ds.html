Introduction to Data Structures

Data structures are fundamental to computer science. They are ways of organizing data in a computer so that it can be used efficiently.

Common Data Structures:

  • Arrays
  • Linked Lists (Singly, Doubly, Circular)
  • Stacks
  • Queues
  • Trees (Binary, AVL, Red-Black)
  • Graphs
  • Hash Tables
